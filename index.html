Downloading Photos from a Facebook Album

I wanted to download this album from Sarawut Intarob to use his pictures as wallpapers.

I didn't want to install a browser extension so I decided to digg the Graph API, and figure out if it was possible to built a tool with front-end technologies only.

Clicking on an image will start its download.
